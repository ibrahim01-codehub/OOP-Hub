Inheritance

Inheritance lets one class (the derived class) acquire the attributes and methods of another class (the base class), then add or override behavior of its own. It's how OOP models "is-a" relationships and avoids rewriting shared logic in every class that needs it.

💡

A derived class only ever adds to or overrides what it inherits — it never removes access the base class already granted.

What You'll Learn Here

  • How a derived class inherits from a base class
  • How to add new members on top of what's inherited
  • How multiple classes can share one common base
  • Why inheritance models real-world hierarchies naturally

Why Inheritance?

  • Avoids duplicating shared logic across similar classes
  • Models natural hierarchies — an "is-a" relationship
  • New behavior can be added without touching the base class
  • Sets up the base-class pointers polymorphism relies on

How It Works

A derived class lists its base class after a colon. It automatically gets every public and protected member the base class defines, and can add new members or override existing ones.

Animal eat(), sleep()
Dog + bark()
Bird + fly()

Syntax

syntax inheritance.cpp 
class Base {
public:
    void baseMethod() { /* shared behavior */ }
};
 
class Derived : public Base {
public:
    void derivedMethod() { /* new behavior */ }
};

Code Example (C++)

example animal.cpp 
#include <iostream>
using namespace std;
 
class Animal {
public:
    void eat() { cout << "This animal eats food." << endl; }
};
 
class Dog : public Animal {         // Dog "is-a" Animal
public:
    void bark() { cout << "Woof!" << endl; }
};
 
int main() {
    Dog rex;
    rex.eat();      // inherited from Animal
    rex.bark();      // defined in Dog
    return 0;
}
Output 
This animal eats food.
Woof!

Pros of Inheritance

  • Reduces code duplication across related classes
  • Shared changes only need to be made once, in the base class
  • Makes real-world hierarchies easy to represent in code
  • Lays the groundwork for polymorphism

Real-Life Analogy

👨‍👦

A child inherits traits from a parent. They get general traits automatically — like having two arms — but can also have traits of their own the parent never had.
